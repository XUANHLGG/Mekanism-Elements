# Version 3.0.0

## Initial NeoForge Conversion

This version marks the conversion of Mekanism: Elements from Forge to NeoForge.

### Changes
- **Port to NeoForge**: Complete migration from Minecraft Forge to NeoForge
- **Minecraft Version**: Updated to support Minecraft 1.21.1
- **Mekanism Compatibility**: Updated to work with Mekanism 10.7.17+

### Technical Notes
- All mod systems have been updated to use NeoForge APIs
- Version management system improved with dynamic version range construction
